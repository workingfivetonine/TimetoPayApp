# TimetoPay (Receipt Tracker) — Brand Kit

Everything below is extracted from the pitch deck stylesheet (`artifacts/pitch-deck/src/index.css`) and reorganized so you can recreate it as a **Canva Brand Kit** (Brand Hub → Brand Kits → add colors & fonts).

---

## 1. Brand Colors

Add these under **Brand Kit → Brand colors**. In Canva, click the color tile, choose **+ Add a new color**, and paste the hex code (without spaces).

### Primary palette
| Swatch name | Hex | Role / where to use |
|---|---|---|
| Violet (Primary) | `#7C3AED` | Main brand color — buttons, links, key highlights, logo |
| Deep Violet | `#5B21B6` | Bookend/cover backgrounds, headers, gradient pair with Primary |
| Light Violet (Accent) | `#A78BFA` | Secondary accents, icons, small highlights, charts |
| Soft Violet (Accent Soft) | `#EDE9FE` | Tints, pill/badge backgrounds, subtle section fills |

### Neutrals
| Swatch name | Hex | Role / where to use |
|---|---|---|
| Surface White | `#FFFFFF` | Cards, panels, content surfaces |
| Page Background | `#FAF8FF` | Slide/page background (off-white with a violet tint) |
| Ink (Text) | `#1E1B2E` | Headlines and primary body text |
| Muted Text | `#6B6385` | Captions, secondary text, labels |

### Supporting / status
| Swatch name | Hex | Role / where to use |
|---|---|---|
| Save Green | `#059669` | "Savings" callouts, positive/success values, price drops |

## 2. Fonts

Add these under **Brand Kit → Brand fonts** (Heading / Subheading / Body text).

| Brand role | Font | Canva availability | Suggested weights |
|---|---|---|---|
| Headings / display | **Sora** | ✅ In Canva's font library — search "Sora" | Bold (700), SemiBold (600) |
| Body / paragraphs | **Manrope** | ✅ In Canva's font library — search "Manrope" | Regular (400), Medium (500), SemiBold (600) |

Both Sora and Manrope are Google Fonts that ship in Canva's standard library (free and Pro), so they match the pitch deck exactly — no substitution needed.

### Fallbacks (only if a font ever isn't available on your plan)
- **Instead of Sora (headings):** *Poppins*, *Montserrat*, or *Space Grotesk* — all geometric, modern, and in Canva.
- **Instead of Manrope (body):** *Inter*, *DM Sans*, or *Work Sans* — all clean, highly legible, and in Canva.

### Type hierarchy (recommended starting points)
| Element | Font | Weight | Notes |
|---|---|---|---|
| Hero / cover title | Sora | Bold 700 | Tightest, largest |
| Slide / section headline | Sora | SemiBold 600 | |
| Subheading | Manrope | SemiBold 600 | |
| Body text | Manrope | Regular 400 | |
| Caption / label | Manrope | Medium 500 | Use Muted Text `#6B6385` |

## 3. Suggested color combinations

- **Cover / bookend slides:** Deep Violet `#5B21B6` background, white text, Light Violet `#A78BFA` accents.
- **Content slides:** Page Background `#FAF8FF` with white cards `#FFFFFF`, Ink `#1E1B2E` text, Violet `#7C3AED` for emphasis.
- **Savings / numbers:** Save Green `#059669` on white or soft violet.
- **Pills & badges:** Soft Violet `#EDE9FE` fill with Violet `#7C3AED` or Deep Violet text.

---

## 4. How to set this up in Canva

1. Open **Canva → Brand Hub → Brand Kits** (Pro feature; free accounts get one limited Brand Kit).
2. Under **Brand colors**, create a palette named "TimetoPay" and add each hex code above.
3. Under **Brand fonts**, set **Sora** as your Heading font and **Manrope** as your Body/Default text font.
4. (Optional) Upload your logo under **Logos**.
